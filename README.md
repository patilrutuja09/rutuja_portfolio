# Rutuja Portfolio

A modern and visually attractive personal portfolio website built with HTML, CSS, and JavaScript featuring a Dark Luxury Purple + Neon Cyan theme.

This portfolio showcases projects, skills, coding profiles, internships, and contact details with smooth animations and a premium futuristic UI.

---

## ✨ Features

- 🌌 Dark Luxury UI Design
- 💎 Neon Cyan Glow Effects
- 📱 Fully Responsive Layout
- ⚡ Smooth Scrolling Navigation
- 🎨 Animated Hero Section
- 🛠️ Skills Showcase
- 🚀 Interactive Project Cards
- 📅 Internship Timeline
- 💻 Coding Profile Section
- 📬 Contact Form
- ✨ Modern Hover Animations
- 🔥 Clean & Professional Design
